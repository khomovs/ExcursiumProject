## Структура проекта
- `conftest.py`: Настройка фикстур для тестов.
- `locators.py`: Локаторы для элементов страниц.
- `pages/`: Классы для страниц (Page Object Model).
- `tests/`: Автоматизированные тесты.
- `requirements.txt`: Зависимости проекта.

## Используемые инструменты
- **Selenium**: Для автоматизации тестирования веб-интерфейса.
- **Pytest**: Для написания и запуска тестов.
- **Allure**: Для генерации отчётов.
